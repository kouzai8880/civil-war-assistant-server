# 错误码汇总

服务器基础URL: https://dvmxujshaduv.sealoshzh.site

## 通用错误码 (1000-1999)

| 错误码 | 描述 | HTTP状态码 |
|-------|------|------------|
| 1001 | 参数错误或字段验证失败 | 400 |
| 1002 | 未认证或认证已过期 | 401 |
| 1003 | 权限不足 | 403 |
| 1004 | 资源不存在 | 404 |

## 错误响应格式

所有错误响应都遵循以下格式:

```json
{
  "status": "error",
  "message": "错误描述信息",
  "code": 1001,
  "errors": [
    {
      "field": "字段名",
      "message": "字段错误描述"
    }
  ]
}
```

其中，`errors` 字段仅在表单验证错误等情况下返回，用于描述具体的字段错误。
